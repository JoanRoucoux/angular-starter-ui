# Typography

The library does not ship fonts: it relies on the host's system font stack, applied on `body` by the consuming
application (Storybook does the same in `.storybook/preview.css`):

```css
font-family:
  ui-sans-serif,
  system-ui,
  -apple-system,
  'Segoe UI',
  sans-serif;
```

## Radius

Two radii are used across the system: `rounded-md` (buttons, inputs, cards) and `rounded-full` (badges, avatars).
